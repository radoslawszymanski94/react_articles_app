Opis aplikacji:
Aplikacja articles-app pozwala na tworzenie, usuwanie oraz edycję artykułów. Wszelkie czynności dotyczące artykułów odkładane są w bazie danych (MongoDB).

Stack technologiczny:

FRONTEND
- React
- Redux
- React Router DOM
- Styled Components
- Axios
- Ant
- Framer Motion
- Moment


BACKEND
- Node JS
- Express
- Axios
- Mongoose
- Dotenv
